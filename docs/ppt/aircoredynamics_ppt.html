🏭 AirCore Dynamics

통합 제조 대시보드

Three.js 기반의 3D 공장 뷰어와 Figma를 활용한 UI/UX 설계를 통해, 데이터 시각화, 예측 분석, 공정 모니터링을 하나의 통합 플랫폼에서 제공하는 웹 기반 스마트 팩토리 대시보드입니다.

✨ 01. 통합 대시보드: 정상 가동 현황

  • 생산 현황: 금일 생산량, 시간대별 추세, 목표 대비 진행률
  • 가동 현황: 전체/공정별 가동률, 가동·정지 장비 수
  • 품질 관리: 검사 통과율, 불량률 및 유형 분포
  • 설비 상태 모니터링: 주요 장비별 상태 (정상/주의/정지)
  • 환경 모니터링: 온도, 습도, 전력 소비량 데이터
  • 알림 패널: 장비 이상, 품질 문제 등 이벤트 알림
  • 3D 공장 뷰어 연동: 데이터와 실제 설비 상태 직관적 연결

✨ 02. 통합 대시보드: 이상 신호 감지 및 대응

  • 알림 패널 강조: 장비 이상, 품질 문제 등 이벤트 발생 시 즉각적으로 강조 표시됩니다.
  • 3D 모델 연동: 이상이 발생한 설비의 3D 모델과 관련 지표가 시각적으로 변화하여 문제 위치를 즉시 파악할 수 있습니다.
  • 신속 대응 지원: 관리자가 즉시 상황을 인지하고 조치할 수 있도록 유도하여 다운타임을 최소화합니다.

✨ 03. 공정별 맞춤 분석: 주조 라인

라인 현황 지표 제공

시간당 생산량, 가동률, 전력 사용량, 불량률 및 주요 설비(주조 온도, 압력 등) 데이터 심층 분석

라인별 3D 뷰 연동

해당 라인의 장비 배치를 3D로 시각화하고, 각 설비 클릭 시 세부 상태 확인

상태 색상 표시

  • 🟢 초록색: 모든 장비 정상 가동
  • 🟡 노란색: 일부 장비만 가동
  • 🔴 빨간색: 모든 장비 정지

✨ 04. 공정별 맞춤 분석: 가공 라인

가공 라인의 실시간 생산량, 가동률, 전력량, 불량률 및 CNC 스핀들 속도, 절삭유량 등 핵심 지표를 제공합니다.

해당 라인의 3D 장비 배치와 가동 상태를 시각적으로 보여주어 직관적인 모니터링이 가능합니다.

✨ 05. 공정별 맞춤 분석: 검사 라인

검사 라인의 생산 현황, 가동률, 전력량, 불량률 등의 데이터를 제공하여 품질 관리에 집중합니다.

검사 장비와 포장 장비의 3D 배치를 통해 공정 흐름을 한눈에 파악할 수 있습니다.

✨ 06. 공정별 맞춤 분석: 조립 라인

조립 라인의 생산량, 가동률, 전력량, 불량률 등 핵심 성과 지표를 실시간으로 모니터링합니다.

조립 장비의 3D 시각화를 통해 장비별 상태를 직관적으로 확인할 수 있습니다.

✨ 07. 공정별 맞춤 분석: 포장/출하 라인

포장/출하 라인의 생산량, 가동률, 전력량, 불량률 등을 종합적으로 관리합니다.

포장 장비의 3D 뷰를 통해 최종 출하 단계의 흐름을 파악하고 최적화할 수 있습니다.

✨ 08. 데이터를 통한 미래 예측 분석

생산량 예측 분석

과거 데이터를 기반으로 미래 생산량을 예측하여 생산 계획 수립을 지원합니다.

병목 구간 분석

주요 병목 공정을 식별하고 평균 대기 시간 및 처리 속도를 분석하여 비효율적인 요소를 개선합니다.

✨ 09. 실시간 재고 및 출하 관리

현재 재고

완제품 및 반제품의 실시간 재고 현황을 시각적으로 파악하고, 재고 현황판을 통해 관리합니다.

출하 현황

금일 출하량, 대기 중인 출하량, 예정된 출하량을 통합 관리하여 물류 흐름을 최적화합니다.

✨ 10. 지능형 비가동 설비 관리

장비별 상태 표시

비가동 중인 장비의 공정, 상태 (정지), 최종 업데이트 시각을 한눈에 파악합니다.

오류 코드 선택 및 서버 전송

비가동 장비를 클릭하면 오류 코드 선택 UI를 제공하여 유지보수팀에 신속하게 정보를 전달할 수 있습니다.

📐 시스템 화면 흐름도 - 단계별 설명

1단계: 메인 대시보드

전체 가동률, 생산량 등 공장 전체 현황을 표시합니다. 공장 이미지를 더블 클릭하면 2단계로 이동합니다.

2단계: 공정별 상세 화면

주조, 가공 등 각 공정별 상세 데이터를 표시합니다. 특정 공정을 더블 클릭하면 3단계로 이동합니다.

3단계: 장비 상세 화면

개별 장비의 상태, 전력량, 생산량 등 상세 정보를 표시합니다. 장비를 클릭하여 정보를 전환합니다.

🖌️ UI/UX (Design - Figma)

사용자 여정(User Flow) 중심의 와이어프레임, 정보 계층 구조를 반영한 지표 시각화, 다크 모드 기반의 현대적 UI를 고려하여 Figma로 설계했습니다.

🏗️ 시스템 아키텍처

Presentation Layer

Vue.js 3 + Three.js 기반 사용자 인터페이스 및 3D 시각화

Business Logic Layer

Spring Boot 기반 REST API 서버, 비즈니스 로직 처리

Data Access Layer

MyBatis를 통한 데이터베이스 접근 및 ORM 매핑

Data Layer

MySQL 데이터베이스, 생산 데이터 및 시계열 데이터 저장

🚀 시작하기 (Getting Started)

# 1. 프로젝트를 클론합니다.

git clone https://github.com/your-repo/Aircore-Dynamics.git
cd Aircore-Dynamics/frontend

# 2. 의존성을 설치합니다.

npm install

# 3. 개발 서버를 실행합니다.

npm run dev
